Проект использует CMake для сборки. 

## 📊 Реализованные алгоритмы

В проекте реализованы следующие алгоритмы сортировки в шаблонном классе `Sorter<T>`:

### Простые квадратичные алгоритмы (O(n²)):
- `bubble_sort` - классическая пузырьковая сортировка
- `bubble_sort_opt` - оптимизированная пузырьковая
- `insertion_sort` - сортировка вставками
- `insertion_sort_shift` - сдвиговая версия
- `insertion_sort_bin` - с бинарным поиском
- `selection_sort` - сортировка выбором

### Улучшенные алгоритмы (O(n log² n)):
- `shell_sort` - сортировка Шелла (последовательность N/2)
- `shell_sort_knuth_arr` - Шелла с последовательностью Кнута (массив)
- `shell_sort_knuth_func` - Шелла с последовательностью Кнута (функция)
- `shell_sort_hibbard_arr` - Шелла с последовательностью Хиббарда (массив)
- `shell_sort_hibbard_func` - Шелла с последовательностью Хиббарда (функция)
- `shell_sort_sedgewick` - Шелла с последовательностью Седжвика
- `heap_sort` - пирамидальная сортировка
- `quick_sort` - базовая версия быстрой сортировки с выбором последнего элемента как pivot
- `quick_sort_opt` - модифицированная реализация алгоритма с выбором медианы трёх
- `quick_sort_safe` - защищённая версия с fallback на HeapSort при глубокой рекурсии
- `merge_sort` - рекурсивная сортировка слиянием с единым временным буфером.

### Алгоритмы внешней сортировки:
- `external_sort_method1`- ES1 - алгоритм внешней сортировки Bucket Sort - создание T разных файлов с последующим слиянием
- `external_sort_method2`- ES2 - алгоритм внешней сортировки Balanced Two-Way Merge Sort - сортировка с двумя вспомогательными файлами
- `external_sort_method3`- ES3 - алгоритм внешней сортировки Block Sort - при первом проходе загрузка блоков в память, внутренняя сортировка, затем слияние по алгоритму ES2

## 🧪 Система тестирования

Класс `Test<T>` обеспечивает:
- Автоматическую загрузку тестовых данных из файлов
- Замер времени выполнения алгоритмов
- Проверку корректности сортировки
- Сохранение результатов в файлы (опционально)
- Пропуск слишком больших тестов для медленных алгоритмов
<br>
Класс `ExternalTest` (для алгоритмнов внешней сортировки) обеспечивает:
- Генерацию тестовых данных
- Замер времени выполнения алгоритмов
- Проверку корректности сортировки
- Пропуск слишком больших тестов для медленных алгоритмов

## 📈 Результаты тестирования

Полный сравнительный анализ представлен в отчетах `report/report_part1.md`, `report/report_part2.md`, `report/report_part3.md` включая:
- Таблицы производительности для разных размеров данных
- Анализ адаптивности алгоритмов
- Влияние типа данных на скорость
- Рекомендации по выбору алгоритма

## 📝 Примечания

- Для экономии места файлы результатов создаются только для демонстрационных целей
- Медленные алгоритмы автоматически пропускают большие тесты
